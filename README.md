# React-based Online Store 🛍️

Современный и отзывчивый интернет-магазин, разрабатываемый на базе библиотеки **React** и инструмента сборки **Vite**.

---

## 📌 Описание проекта
Этот проект представляет собой прототип и будущую полноценную реализацию современного веб-приложения для онлайн-торговли. Основная цель — создать быстрый, удобный и интуитивно понятный интерфейс для пользователей с использованием лучших практик разработки на React.

---

## 🛠️ Текущая структура проекта
На данный момент в репозитории развернута базовая архитектурная структура приложения. Проект разделен на логические слои для обеспечения масштабируемости и поддерживаемости кода:

```text
react/
├── public/                 # Статические ресурсы (логотипы, иконки)
├── src/
│   ├── components/         # Компоненты интерфейса
│   │   ├── common/         # Общие компоненты (Header, Footer, Button)
│   │   ├── product/        # Компоненты для работы с товарами (ProductCard, ProductList)
│   │   └── cart/           # Компоненты корзины (CartItem)
│   ├── context/            # Глобальное состояние приложения (CartContext)
│   ├── hooks/              # Пользовательские хуки (useLocalStorage)
│   ├── pages/              # Страницы приложения (Home, ProductDetail, Cart, Checkout, NotFound)
│   ├── services/           # Сервисы и API-клиенты (mockProducts)
│   ├── App.jsx             # Корневой компонент приложения
│   ├── index.css           # Глобальные стили
│   └── main.jsx            # Точка входа в React-приложение
├── index.html              # HTML-шаблон
├── package.json            # Зависимости и скрипты проекта
└── vite.config.js          # Конфигурация Vite
```

## 🚀 Планы на будущее (Roadmap)

В процессе разработки планируется реализовать следующий функционал:

### 1. Функциональные возможности (Features)
- **Каталог товаров:** Вывод списка товаров с фильтрацией по категориям, сортировкой (по цене, популярности) и поиском.
- **Интерактивная корзина:** Добавление/удаление товаров, изменение количества, автоматический пересчет стоимости заказа.
- **Детальная страница товара:** Просмотр подробного описания, характеристик и отзывов.
- **Процесс оформления заказа (Checkout):** Форма для заполнения контактных данных, интеграция валидации и симуляция отправки заказа.
- **Локальное хранилище (Local Storage):** Автоматическое сохранение содержимого корзины между сессиями пользователя.
- **Панель избранного (Wishlist):** Возможность откладывать понравившиеся товары.

### 2. Технические улучшения
- **Роутинг:** Интеграция `react-router-dom` для бесшовной навигации между страницами (Каталог, Корзина, Детали товара, Оформление).
- **Стилизация:** Разработка современного UI с использованием CSS-переменных, адаптивной верстки (Mobile First) и плавных анимаций.
- **Управление состоянием:** Расширение контекста (React Context API) или внедрение стейт-менеджеров для обработки сложных состояний.
- **Имитация API:** Создание полноценного слоя сервисов для работы с mock-данными или интеграция с внешним публичным REST API (например, FakeStoreAPI).

## 💻 Запуск проекта локально

### Требования
- Установленный [Node.js](https://nodejs.org/) (версии 18.x или выше)
- Менеджер пакетов `npm` или `yarn`

### Инструкция по запуску
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Voidhmslf/React-based-online-store.git
   cd React-based-online-store
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```
4. Откройте приложение в браузере по адресу: [http://localhost:5173](http://localhost:5173)
